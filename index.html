    модальне вікно №2



    HTML (HyperText Markup Language) — це мова розмітки, яка використовується для створення структури веб-сторінок. Вона визначає, як елементи, як-от заголовки, абзаци, зображення та посилання, розміщуються на сторінці.

    let має блочну область видимості і не піддається підняттю (hoisting) так, як var. Використання let вважається сучаснішим та безпечнішим способом оголошення змінних.

    React — це JavaScript-бібліотека для створення інтерфейсів користувача. Вона дозволяє створювати повторно використовувані компоненти та керувати станом застосунку ефективно.

    Git — це система контролю версій, яка дозволяє розробникам відслідковувати зміни в коді, працювати над проектом у команді та легко повертатись до попередніх версій у разі помилок.
